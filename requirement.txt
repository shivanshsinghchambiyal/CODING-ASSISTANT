## Libraries Used

### PyAutoGUI
Used for screen automation and capturing screenshots. It enables the assistant to analyze content directly from the user's screen.

### PyTesseract
Python wrapper for Tesseract OCR. It extracts text from screenshots, allowing the assistant to understand coding problems and compiler errors.

### Pillow (PIL)
Used for image processing and handling screenshot files before OCR extraction.

### PyGetWindow
Used to detect and access specific application windows (such as LeetCode in Chrome), making screenshot capture more accurate.

### Tesseract OCR
An open-source Optical Character Recognition engine used to convert images into machine-readable text.
